# House Price Prediction Project

## Project Overview

This is a Machine Learning project used to predict house prices using Python and Linear Regression.

The main goal of this project is to understand data cleaning, data analysis, model training, and prediction using real-world housing data.

---

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook
* GitHub

## Project Steps

1. Import Libraries

4. Handle Missing Values
5. Exploratory Data Analysis

9. Prediction

11. Save Model
